Movie Picker

Full-Stack Machine Learning Project.

Click to visit the project page

The goal of this project was to make the simplest possible movie recommendation tool. There's no login required, and it features Machine Learning in the Back-End to find the best movie for you.

Just like or dislike at least ten different movies to unlock your recommendations!

The building process was very fun, and included Web Scraping movie information from IMDB with Python (Beautiful Soup and Selenium). The movie information and user votes are store in a SQLite database, and you can vote as many times as you want without making the algorithm biased. The machine learning part makes use of the Python's Scikit-learn library, and the whole project is served using the Flask library.

The Front-End makes use of Boostrap and jQuery.

  • Date: January 2018

Personal Finance

Full-Stack project with login system.

Click to visit the project page

This is a Personal Finance planner and tracker, and it was my first attempt at Full-Stack project. It's a single page app, and it features a homemade login system.

In the Front-End everything was made 'by hand' using only jQuery, as I wanted to learn as much as possible in the process. It asynchronously interacts with the Back-End API without reloading the page.

The Back-End is again Python. It stores all the user login, preferences, and expense information in a SQLite database.

The login system is a simple implementation designed by myself (it's not to be trusted, especially since the page is not using https), and the user's password are not stored raw in my system (the Front-End sends it already SHA-256 hashed to the Back-End API).

This project was hugely educational, and although it could be much improved, I still use it everyday to keep track of my own expenses.

  • Date: December 2017
