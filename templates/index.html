Skills

Technologies I have worked with

Python

I'm quite familiar with Python, having used it for many different purposes from game bots to scientific computing (Numpy, Matplotlib, Scipy, Scikit-Learn etc.). I also host all my web pages (including this one) using the Python's Flask library.

JavaScript

I have been using JavaScript ever since I started my Udacity's Front-End Nanodegree. I love the new ES6 improvements, and I want to start playing with Node.js soon!

HTML/CSS

HTML5 and CSS3 are of course a must for any Front-End web developer, and are also part of the technologies that I deal with everyday.

SQLite

I have used SQLite for more projects than I can remember. I love how simple it is to use, and I hope it will make my life easier when I decide to learn MySQL and PostgreSQL.

Git

I use Git to keep track of the changes I make to my projects, and it keeps me sane. Thanks Git! I also have a GitHub account linked on the page's footer.

Linux

Although my main machine is Windows based, I have been playing around with the Linux shell for a while. I currently have Linux based Raspberry Pi computers, on which I host webapps and make fun automation projects.

jQuery

jQuery lost a bit of it's charm now that we have ES6 but I still like it for some things, and used it more often in the past.

Bootstrap

Two of my pages are Bootstrap based (including this one). Although I'm not hugely familiar, I have played around with it a bit.
